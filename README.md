koala-dust
==========

A koala extension that adds support for compiling dust file.

How to build
------------

1. run `npm install`.
2. create a zip file.
3. add, to the zip, all files:
    * node_mudules
    * dust.png
    * DustCompiler.js
    * LICENSE
    * package.js
4. rename zip to `dust.koala-compiler`.

How to install
--------------

Drag-n-Drop `dust.koala-compiler` on Koala, and it should install it.
